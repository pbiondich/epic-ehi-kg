# AGENDA_TOPIC_INFO

> Table of no-add single row information about visit agenda topics (VATs).

**Primary key:** `AGENDA_TOPIC_ID`  
**Columns:** 16  
**Joined by:** 8 columns

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `AGENDA_TOPIC_ID` | NUMERIC | PK | The unique identifier (.1 item) for the topic record. |
| 2 | `CREATION_UTC_DTTM` | DATETIME (UTC) |  | UTC instant that the record was first created |
| 3 | `AGENDA_CODEASSIST_C_NAME` | NUMERIC |  | Indicates the type of agent which created this topic |
| 4 | `LINKED_AGENDA_ID` | NUMERIC |  | Agenda that contains the given topic |
| 5 | `CLIN_TITLE` | VARCHAR |  | The title/name of this topic |
| 6 | `CLIN_DESC` | VARCHAR |  | The description for this topic |
| 7 | `CREATED_BY_USER_ID` | VARCHAR |  | Clinical user who created the topic |
| 8 | `CREATED_BY_USER_ID_NAME` | VARCHAR |  | The name of the user record. This name may be hidden. |
| 9 | `PAT_TITLE` | VARCHAR |  | The topic title/name for a patient topic |
| 10 | `PAT_DESC` | VARCHAR |  | The description for a patient topic |
| 11 | `CREATED_BY_MYPT_ID` | VARCHAR |  | MyChart user who created the topic |
| 12 | `DYN_QUES_ANSWER_ID` | VARCHAR |  | Reference to the answers for questions asked by the dynamic questions agent within MyChart |
| 13 | `DYN_QUES_ORIG_TEXT` | VARCHAR |  | Text entered by the MyChart user which prompted the use of the dynamic questions agent |
| 14 | `PAT_PRIMARY_YN` | VARCHAR |  | Has the patient marked this as their primary topic? |
| 15 | `ASSOCIATED_DX_UID` | VARCHAR |  | The diagnosis this agenda topic was associated with during creation, if applicable. |
| 16 | `AGENT_TOPIC_SUMMARY` | VARCHAR |  | Topic summary or reason for the topic as generated by the AI agent. |
