# LEARN_ASSESS_INFO

> This table contains the basic no-add single items of Learning Assessment template/topic (ILA) records like record name, state, type, level, no-add items edited and instant of their edit, etc.

**Primary key:** `ILA_RECORD_ID`  
**Columns:** 3

[← index](../index.md)

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `ILA_RECORD_ID_ILA_RECORD_NAME` | VARCHAR |  | The name for this Learning Assessment Template/Topic record. |
| 2 | `ILA_RECORD_NAME` | VARCHAR |  | The name for this Learning Assessment Template/Topic record. |
| 3 | `ILA_DISPLAY_NAME` | VARCHAR |  | The display name of the learning assessment record. |
