# PR_EST_COVERAGES

> Multiple response general information about the price estimate. Currently stores multiple coverages applicable on dental estimate.

**Primary key:** `ESTIMATE_ID`, `LINE`  
**Columns:** 3

[← index](../index.md)

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `ESTIMATE_ID` | NUMERIC | PK shared | The unique identifier for the patient estimate record. |
| 2 | `LINE` | INTEGER | PK | The line number for the information associated with this record. Multiple pieces of information can be associated with this record. |
| 3 | `COVERAGE_ID` | NUMERIC | FK→ | Coverage references for multiple coverages on estimate when available. This item is multiple response, and used only for Wisdom and advance bill estimates. |
