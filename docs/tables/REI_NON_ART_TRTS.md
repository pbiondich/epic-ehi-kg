# REI_NON_ART_TRTS

> The REI_NON_ART_TRTS table contains information about the procedures needed for the assisted reproductive technology (ART) treatment type, which could be egg retrieval or embryo transfer.

**Primary key:** `CYCLE_ID`, `LINE`  
**Columns:** 3  
**Org-specific columns:** 1
