# ORDER_CASE_PANEL_INFO

> This table stores data about panels for case request orders.

**Primary key:** `ORDER_ID`, `LINE`  
**Columns:** 9

[← index](../index.md)

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `ORDER_ID` | NUMERIC | PK shared | The unique identifier (.1 item) for the order record. |
| 2 | `LINE` | INTEGER | PK | The line number for the information associated with this record. Multiple pieces of information can be associated with this record. |
| 3 | `REQUESTED_PROC_MINUTES` | INTEGER |  | This item contains the surgeon's requested total procedure length in minutes for a panel in a case request order. |
| 4 | `PROC_LENGTH_MODIFIED_YN` | VARCHAR |  | This item stores whether the requested panel length is a user-modified value. |
| 5 | `REQUESTED_PROC_COMMENTS` | VARCHAR |  | This item contains the surgeon's comments about a requested total procedure length for a panel in a case request order. |
| 6 | `PANEL_CONFIDENCE_C_NAME` | VARCHAR |  | Confidence score for the estimated time for a panel generated by Epic |
| 7 | `PAN_DURATION_ESTIMATE_MINUTES` | INTEGER |  | Estimated panel duration generated by Epic |
| 8 | `PANEL_REQ_LEN_USER_ID` | VARCHAR |  | This item tracks the last user to modify the requested total procedure length in minutes for a panel in a case request order so that the default length is not used. |
| 9 | `PANEL_REQ_LEN_USER_ID_NAME` | VARCHAR |  | The name of the user record. This name may be hidden. |

_Flags: PK = primary key · org = may contain organization-specific values · discont. = discontinued · FK→ = inferred reference (see below) · shared = generic key, intentionally unresolved._
