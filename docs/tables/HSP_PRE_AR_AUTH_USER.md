# HSP_PRE_AR_AUTH_USER

> This table extracts the auth override user information associated with hospital billing temporary transactions. This table is limited to charge temporary transactions that have not yet been posted to the account.

**Primary key:** `HTT_ID`, `GROUP_LINE`, `VALUE_LINE`  
**Columns:** 5

[← index](../index.md)
