# PAT_PHENO

> This table contains information about a patient's phenotypes as they are documented over time.

**Primary key:** `PAT_PHENO_ID`, `CONTACT_DATE_REAL`  
**Columns:** 15  
**Org-specific columns:** 1

[← index](../index.md)
