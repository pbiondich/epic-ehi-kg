# HMA_DESCRIPTION

> This table holds relevant metadata about user macros. This includes SmartBlock macros and SmartForm macros, differentiated by type (I HMA 70).

**Primary key:** `RECORD_ID`  
**Columns:** 3

[← index](../index.md)

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `RECORD_ID` | NUMERIC | PK shared | Stores the ID of the macro. |
| 2 | `RECORD_ID_RECORD_NAME` | VARCHAR |  | Stores the name of a macro. |
| 3 | `RECORD_NAME` | VARCHAR |  | Stores the name of a macro. |

_Flags: PK = primary key · org = may contain organization-specific values · discont. = discontinued · FK→ = inferred reference (see below) · shared = generic key, intentionally unresolved._
