# IMG_LOAD_BALANCE_READ_SER

> The table storing the data in the related multi-response ORD Item 52872. This data represents the reading providers responsible for an order at a given instant a load balancing action was performed on the order.

**Primary key:** `ORDER_ID`, `GROUP_LINE`, `VALUE_LINE`  
**Columns:** 4
