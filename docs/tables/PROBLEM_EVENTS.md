# PROBLEM_EVENTS

> This table contains a list of relevant events documented as the history of a patient's problem.

**Primary key:** `PROBLEM_LIST_ID`, `CONTACT_DATE_REAL`, `LINE`  
**Columns:** 20  
**Org-specific columns:** 2

[← index](../index.md)

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `PROBLEM_LIST_ID` | NUMERIC | PK FK→ | The unique identifier for the problem record. |
| 2 | `CONTACT_DATE_REAL` | FLOAT | PK | A unique contact date in decimal format. The integer portion of the number indicates the date of contact. The digits after the decimal distinguish different contacts on the same date and are unique for each contact on that date. For example, .00 is the first/only contact, .01 is the second contact, etc. |
| 3 | `LINE` | INTEGER | PK | The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact. |

| 7 | `PRB_EVENT_HX_TYPE_C_NAME` | VARCHAR | org | Stores the history type that each event was created from |
| 8 | `PRB_EVENT_STDATE_DT` | DATETIME |  | Stores the start date of each event |
| 9 | `PRB_EVENT_ENDATE_DT` | DATETIME |  | Stores the end date of each event |
| 10 | `PRB_EVENT_CAT_C_NAME` | VARCHAR | org | Stores the category/type of each event |
| 11 | `PRB_EVENT_DESC` | VARCHAR |  | Stores the description/details of each event |
| 12 | `PRB_EVENT_NOTE_ID` | VARCHAR |  | Stores the problem event note ID |
| 13 | `PROBLEM_EVENT_LINKED_ACTION_C_NAME` | VARCHAR |  | This item contains the action configured in Problem List Automated Actions (I LSD 2765) that triggered the automatic creation of the oncology history event on the same line. |
| 14 | `PROBLEM_EVENT_AUTO_UPDATE_YN` | VARCHAR |  | This item stores a yes or no to indicate whether or not an event will automatically update. |

| 16 | `PRB_EVENT_USR_DTL_AUTO_NOTE_ID` | VARCHAR |  | This column contains the note ID for the note containing the auto-generated event's user-entered comments. |

| 18 | `PRB_EVENT_EXTERNAL_UNIQUE_KEY` | VARCHAR |  | This item contains a key to uniquely identify an oncology history event created by the external events API. If filing an automated event based on chronicles data, this item should not be used. |
| 19 | `PROB_EXTERNAL_EVENT_START_DATE` | DATETIME |  | This item contains the start date for an oncology history event filed from the external events API. It should contain the same value as LPL 8103-problem event start date unless a user has chosen to manually override the start date for the event. If filing an automated event based on chronicles data, this item should not be used. |
| 20 | `PROB_EXTERNAL_EVENT_END_DATE` | DATETIME |  | This item contains the end date for an oncology history event filed from the external events API. It should contain the same value as LPL 8104-problem event end date unless a user has chosen to manually override the end date for the event. If filing an automated event based on chronicles data, this item should not be used. |
