# NHSN_DEFINITION

> This table holds information about National Healthcare Safety Network (NHSN) definition records. The records included in this table are Regulatory Group records that are designated as NHSN Location or NHSN Facility records.

**Primary key:** `NHSN_DEF_ID`  
**Columns:** 2

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `NHSN_DEF_ID_CMS_MU_NAME` | VARCHAR |  | The name of the CMS Meaningful Use record. |
| 2 | `NHSN_DEF_NAME` | VARCHAR |  | The name of the National Healthcare Safety Network (NHSN) definition record. |
