# FIXED_FORMAT_INFO

> All values associated with a claim are stored in the Claim External Value record. The FIXED_FORMAT_INFO table holds additional information not accounted for by other fields on the claim.

**Primary key:** `RECORD_ID`, `LINE`  
**Columns:** 3

[← index](../index.md)

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `RECORD_ID` | NUMERIC | PK shared | The unique identifier for the claim record. |
| 2 | `LINE` | INTEGER | PK | The line number of one of the multiple values associated with a specific group of data within this record. |
| 3 | `FIXED_FORMAT_INFO` | VARCHAR |  | This item holds additional information required for the claim that cannot be submitted in other fields. |

_Flags: PK = primary key · org = may contain organization-specific values · discont. = discontinued · FK→ = inferred reference (see below) · shared = generic key, intentionally unresolved._
