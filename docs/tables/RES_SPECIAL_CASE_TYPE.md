# RES_SPECIAL_CASE_TYPE

> This table contains a result's BI-RADS special case type.

**Primary key:** `FINDING_ID`, `LINE`  
**Columns:** 3  
**Org-specific columns:** 1

| # | Column | Type | Flags | Description |
