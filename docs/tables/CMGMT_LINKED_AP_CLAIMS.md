# CMGMT_LINKED_AP_CLAIMS

> This table contains a list of AP Claims linked to Coordinated Care Management episodes.

**Primary key:** `SUMMARY_BLOCK_ID`, `LINE`  
**Columns:** 3

[← index](../index.md)

| # | Column | Type | Flags | Description |

| 1 | `SUMMARY_BLOCK_ID` | NUMERIC | PK shared | The unique identifier for the episode record. |

| 3 | `LINKED_AP_CLAIM_ID` | NUMERIC |  | This column displays the identifiers of AP claims linked to a care management episode. |
