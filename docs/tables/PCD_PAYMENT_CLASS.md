# PCD_PAYMENT_CLASS

> The PCD_PAYMENT_CLASS contains information about your payment classification definition records. The records included in this table are records from the PCD master file.

**Primary key:** `PCD_ID`  
**Columns:** 3

[← index](../index.md)

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `PCD_ID` | NUMERIC | PK | The unique identifier for the Payment Classification Definition (PCD) record. |
| 2 | `PCD_ID_PCD_NAME` | VARCHAR |  | The group title of the payment classification. |
| 3 | `PCD_NAME` | VARCHAR |  | The group title of the payment classification. |

_Flags: PK = primary key · org = may contain organization-specific values · discont. = discontinued · FK→ = inferred reference (see below) · shared = generic key, intentionally unresolved._
