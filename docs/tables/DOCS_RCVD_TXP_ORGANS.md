# DOCS_RCVD_TXP_ORGANS

> This is the table for transplanted organs that will be linked to the transplant table through a reference Id.

**Primary key:** `DOCUMENT_ID`, `CONTACT_DATE_REAL`, `LINE`  
**Columns:** 19  
**Org-specific columns:** 1

[← index](../index.md)

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `DOCUMENT_ID` | NUMERIC | PK shared | The unique identifier (.1 item) for the document record. |
| 2 | `CONTACT_DATE_REAL` | FLOAT | PK | A unique contact date in decimal format. The integer portion of the number indicates the date of contact. The digits after the decimal distinguish different contacts on the same date and are unique for each contact on that date. For example, .00 is the first/only contact, .01 is the second contact, etc. |
| 3 | `LINE` | INTEGER | PK | The line number for the information associated with this contact. Multiple pieces of information can be associated with this contact. |
| 4 | `CONTACT_DATE` | DATETIME |  | The date of this contact in calendar format. |
| 5 | `TRANSPLANT_REFERENCE_IDENT` | VARCHAR |  | This stores the unique reference id associated with the transplant episode that is a reference to the related group line DXR 11700 |
| 6 | `TRANSPLANT_ORGAN_IDENT` | VARCHAR |  | This stores the reference id generated by the organ ID linked to the episode. |
| 7 | `TRANSPLANTED_ORGAN_TYPE_ID` | NUMERIC |  | Organ type for the transplanted organ. |
| 8 | `TRANSPLANTED_ORGAN_TYPE_ID_RECORD_NAME` | VARCHAR |  | Stores record name (.2) |
| 9 | `TRANSPLANT_DONOR_SOURCE_TYPE_C_NAME` | VARCHAR | org | Transplanted organ's donor type. |
| 10 | `TRANSPLANT_ORGAN_KDPI_SCORE` | INTEGER |  | Specifies the kidney donor's risk profile percentage. Associated with the received kidney |
| 11 | `TXP_ORGAN_COLD_ISCHC_TIME_1` | INTEGER |  | Organ cold ischemic time calculated based on number of minutes between clamp on to clamp off. |
| 12 | `TXP_ORGAN_COLD_ISCHC_TIME_2` | INTEGER |  | Organ cold ischemic time calculated based on number of minutes between clamp on to anastamosis. |
| 13 | `TXP_DONOR_SEROLOGY_CMV_C_NAME` | VARCHAR |  | Organ donor's CMV serology test result. |
| 14 | `TXP_DONOR_SEROLOGY_EBV_C_NAME` | VARCHAR |  | Organ donor's EBV serology test result. |
| 15 | `TXP_DONOR_SEROLOGY_HCV_C_NAME` | VARCHAR |  | Organ donor's HCV serology test result. |
| 16 | `ORGAN_SINGLE_SOURCE_DXO_ID` | NUMERIC |  | Source organization for transplanted organ info when the data is sourced from a single organization. |
| 17 | `ORGAN_SINGLE_SOURCE_DXO_ID_EXTERNAL_NAME` | VARCHAR |  | Organization's external name used as the display name on forms and user interfaces. |
| 18 | `TXP_RECOVERED_DATE` | DATETIME |  | The date that an organ was recovered. |
| 19 | `TXP_DONOR_YN` | VARCHAR |  | This item denotes whether this organ was reported from a donor (ie this patient's EPT ID equals the donor EPT ID) |

_Flags: PK = primary key · org = may contain organization-specific values · discont. = discontinued · FK→ = inferred reference (see below) · shared = generic key, intentionally unresolved._
