# GOAL

> This table contains data on discrete goals (IGO) records associated with a patient.

**Columns:** 13

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `GOAL_ID` | VARCHAR | PK | The unique ID for the goal record. |
| 2 | `GOAL_TYPE_ID` | VARCHAR | FK→ | The unique ID of the goal type associated with this goal. |
| 3 | `GOAL_TYPE_ID_GOAL_NAME` | VARCHAR |  | The name of the goal type record. |
| 4 | `PROBLEM_ID` | VARCHAR | FK→ | The unique ID for the problem linked to this goal. |

| 6 | `DISPLAY_NAME` | VARCHAR |  | This column holds the display name of this goal. |

| 8 | `EVALUATION_NOTE_ID` | VARCHAR |  | This column holds the evaluation note ID for this goal. |

| 10 | `GOAL_TEMPLATE_ID_GOAL_TEMPLATE_NAME` | VARCHAR |  | The goal template name. |
| 11 | `GOAL_TEMPLATE_CDR` | FLOAT |  | This column stores contact date real of the goal template contact associated with this goal, stored in Epic's DTE format. |
| 12 | `HH_EPISODE_ID` | NUMERIC |  | The associated HH/HSPC episode ID for a care plan goal. |
