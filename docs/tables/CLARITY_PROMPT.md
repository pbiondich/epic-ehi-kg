# CLARITY_PROMPT

> This table contains basic information about each Prompt record from the Protocol Prompts (LPQ) master file.

**Primary key:** `PROMPT_ID`  
**Columns:** 3

[← index](../index.md)

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `PROMPT_ID_PROMPT_NAME` | VARCHAR |  | The display name of the prompt record. |
| 2 | `PROMPT_NAME` | VARCHAR |  | The display name of the prompt record. |
| 3 | `DISPLAY_TITLE` | VARCHAR |  | This column stores the title displayed for a prompt in the Nurse Triage Protocol. |

_Flags: PK = primary key · org = may contain organization-specific values · discont. = discontinued · FK→ = inferred reference (see below) · shared = generic key, intentionally unresolved._
