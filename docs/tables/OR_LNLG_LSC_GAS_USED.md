# OR_LNLG_LSC_GAS_USED

> This table contains documentation regarding the gas used when using lasers.

**Primary key:** `RECORD_ID`, `LINE`  
**Columns:** 3  
**Org-specific columns:** 1

| 3 | `LSC_GAS_USED_C_NAME` | VARCHAR | org | This item is used to document whether anesthesia gases were in use during laser usage. |
