# CUST_ID_ADDRESS_STREET

> This table contains the street address lines from the address associated with a customer ID type document.

**Primary key:** `DOCUMENT_ID`, `LINE`  
**Columns:** 3

[← index](../index.md)

| 3 | `RX_CUST_ID_STREET` | VARCHAR |  | The street component of the address from the customer ID. |
