# NESTED_FORM_COMPONENT

> This table contains information about the nested form components used in hemodynamic data entry templates. These include sets, subsets, fields, and field components. This table contains the information about components that does not change overtime.

**Primary key:** `FORM_COMP_ID`  
**Columns:** 2
