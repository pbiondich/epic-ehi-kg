# PRINTERS

> This table contains a listing of Epic printing devices. While these devices primarily are printers, the table can also include devices that are mapped for faxing, EDI interfaces, and other third party uses.

**Primary key:** `PRINT_DEVICE_ID`  
**Columns:** 3

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `PRINT_DEVICE_ID` | NUMERIC | PK | The unique identifier for the device record. |
| 2 | `PRINT_DEVICE_ID_PRINT_DEVICE_NAME` | VARCHAR |  | The printer name. |
| 3 | `PRINT_DEVICE_NAME` | VARCHAR |  | The printer name. |
