# AP_CLAIM_IF_MED_VIS_DXSSF

> This table extracts data received from PPS pricer for a claim in GOB2.APG 'Medical Visit Diagnosis/SSF'.

**Primary key:** `CLAIM_ID`, `GROUP_LINE`, `VALUE_LINE`  
**Columns:** 4

[← index](../index.md)
