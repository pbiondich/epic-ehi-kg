# PROB_LNKSRC_ADM

> This table contains problems from a patient's problem list along with linked admissions suspected of causing the problems.

**Primary key:** `PROBLEM_LIST_ID`, `CONTACT_DATE_REAL`, `LINE`  
**Columns:** 5

[← index](../index.md)

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `PROBLEM_LIST_ID` | NUMERIC | PK FK→ | The unique identifier for the problem record. |
| 2 | `CONTACT_DATE_REAL` | FLOAT | PK | A unique internal contact date in decimal format. The integer portion of the number indicates the date of the contact.
