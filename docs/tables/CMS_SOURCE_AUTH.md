# CMS_SOURCE_AUTH

> This table holds the unique IDs of the service level authorization records where the authorization number was found when the number was found in the authorization record.

**Primary key:** `CLAIM_PRINT_ID`, `GROUP_LINE`, `VALUE_LINE`  
**Columns:** 4
