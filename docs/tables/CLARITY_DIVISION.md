# CLARITY_DIVISION

> This table contains information about your division records. This includes the division's address as well as other division-specific information. The records in this table are Division (DPG) records.

**Primary key:** `DIVISION_ID`  
**Columns:** 3

[← index](../index.md)

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `DIVISION_ID` | VARCHAR | PK | The unique ID of the division record. |
| 2 | `DIVISION_ID_DIVISION_NAME` | VARCHAR |  | The name of the division record. |
| 3 | `DIVISION_NAME` | VARCHAR |  | The name of the division record. |
