# CMGMT_EPISODE_TYPES

> This table contains a list of Coordinated Care Management episode case types. Case types refer to case management programs.

**Primary key:** `EPISODE_ID`, `LINE`  
**Columns:** 3  
**Org-specific columns:** 1

[← index](../index.md)

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `EPISODE_ID` | NUMERIC | PK FK→ | The unique identifier for the summary block record. |
| 2 | `LINE` | INTEGER | PK | The line number for the information associated with this record. Multiple pieces of information can be associated with this record. |
| 3 | `CMGMT_TYPE_C_NAME` | VARCHAR | org | Stores the category ID for the types on the Coordinated Care Management episode. |
