# VISIT_SET_ROUTE_DAYS_WEEK

> This table extracts the related multiple response item VST-10022. Each row in this table represents a day of the week that a provider should be scheduled on for a visit set version. If a visit in this visit set falls on this day of the week, the provider will be requested for the visit.

**Primary key:** `VISIT_SET_ID`, `CONTACT_DATE_REAL`, `GROUP_LINE`, `VALUE_LINE`  
**Columns:** 6
