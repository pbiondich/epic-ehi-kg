# ORDER_PROC_6

> The ORDER_PROC_6 table enables you to report on the procedures ordered in the clinical system. This procedure table has the same basic structure as ORDER_PROC, but was created as a sixth table to prevent ORDER_PROC_5 from getting any larger.

**Overflow of:** [ORDER_PROC](ORDER_PROC.md)  
**Primary key:** `ORDER_ID`  
**Columns:** 88  
**Org-specific columns:** 4

[← index](../index.md)
