# PROBLEM

> This table contains data on care integrator problems associated with a patient.

**Primary key:** `PROBLEM_ID`  
**Columns:** 13  
**Org-specific columns:** 2  
**Joined by:** 34 columns

## Columns

| # | Column | Type | Flags | Description |
|--:|--------|------|-------|-------------|
| 1 | `PROBLEM_ID` | VARCHAR | PK | The unique ID for the care integrator problem. |
| 2 | `TYPE_PROBLEM_ID_NAME` | VARCHAR |  | The name of the multidisciplinary diagnoses. |
| 3 | `CREATED_DATE` | DATETIME |  | The date the problem was added to the patient's care plan. |
| 4 | `DELETED_DATE` | DATETIME |  | The date the problem was removed from the patient's care plan. |
| 5 | `DEL_REASON_C_NAME` | VARCHAR | org | The reason a problem was removed. |
| 6 | `DISCIPLINE_C_NAME` | VARCHAR | org | The discipline associated with this problem. |
| 7 | `IP_DISC_TYPE_ID` | VARCHAR |  | The unique ID for the discipline associated with this problem. |
| 8 | `IP_DISC_TYPE_ID_DISC_NAME` | VARCHAR |  | The name of the discipline record. |
| 9 | `DISPLAY_NAME` | VARCHAR |  | This column shows the record's display name. |
| 10 | `PROBLEM_DESCRIPTION_HNO_ID` | VARCHAR |  | The linked note record for the care plan problem description. This column is frequently used to link to the HNO_INFO table. |
| 11 | `HH_EPISODE_ID` | NUMERIC |  | The associated episode ID for a care plan problem. |
| 12 | `CHILD_LAST_ADDED_DTTM` | DATETIME (Local) |  | The date and time when a goal or intervention was last added to the problem. |
| 13 | `IS_AUTOGENERATED_YN` | VARCHAR |  | This item determines whether a problem is autogenerated. |
